Module: cssdoc

Gera documentação HTML a partir de comentários customizados em arquivos CSS. Os blocos de documentação devem estar entre /*--- e ---*\/, contendo title, section e resume. Lê arquivos CSS em 'public/style' e gera HTML em 'docs/css'.
